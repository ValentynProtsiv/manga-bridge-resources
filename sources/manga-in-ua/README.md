# Manga.in.ua

Metadata package for the Manga.in.ua resource used by MangaBridge.

- Resource id: `manga-in-ua`
- Adapter key: `mangaInUa`
- Language: Ukrainian (`uk`)
- App-side adapter: `features/resources/mangaInUa/`

This package contains metadata only. Request construction, URL resolution, and
parsing remain in the MangaBridge app until resources are distributed as
standalone executable extensions.
